# Maquetacion de Clon de whats app" (comision 999200928)
## maquetacion para proyecto web- entrega de trabajo practico.

Este proyecto es un clon de what's app, un medio de mensajeria instantanea. 

El contenido representa a la planilla principal del usuario, con una lista de contacto y una conversacion abierta.

La maquetacion esta hecha en base de un html y estilado con Css, sin javascript.

## Estructura del proyecto: 

### 1. /INDEX.HTML
### 2. /STYLE/MAIN.CSS
### 3. /README.MD

### Visualizacion: 
Abrir index.html en cualquier navegador.
